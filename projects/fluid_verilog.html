SystemVerilog Fluid Simulation

In this project, I created the most basic Eulerian fluid simulation on the DE10-Lite FPGA board using SystemVerilog. The Gauss-Seidel method was used to ensure continuity of the the flow field. Advection, the change in velocity over time, was not implemented yet in this project. The digital design consists of three different dual-port block RAM layers. The first layer consists of the horizontal and vertical velocities at the edges of each cell, which are updated based on the Gauss-Seidel method and the cursor input. The magnitude and direction of the flow for each cell are then calculated using horizontal and vertical velocities and stored into the second second memory layer. Next, these values are used to update the third layer, which is a framebuffer for the VGA monitor. To draw the arrows, each pixel was projected onto the velocity vector, and if the distance to the vector was less than the line width, the pixel was colored in. This may have not been the most efficient way of drawing an arrow, but it was the most intuitive for me. The Quartus project, SystemVerilog code, and testbenches can be found here.
