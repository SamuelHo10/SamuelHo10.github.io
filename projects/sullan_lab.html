I am very grateful for the opportunity to work with at the Sullan Lab. Everyone was very kind and welcoming, and I learned a lot about the research process.

I was tasked to code an application in python to analyze AFM single-cell force spectroscopy data. I used the Tkinter library for UI and the Pandas library for data manipulation. The application is able to filter out outliers, generate graphs, and format the data on a spreadsheet in a way that is easily readable. The code can be found here.

I also helped with the microfluidics project in the lab. The goal of the project was to create a microfluidics device coated with a metal which is able to help catalyze the degradation of methylene blue. The device was created by casting PDMS, coating the channel with PDA, and then adding a final coat of metal. The surface was then characterized using an AFM to determine if the metal attached onto the PDA successfully. To test out the device, we used a microplate UV-Vis reader to determine concentration of methylene blue before and after passing through the device.
