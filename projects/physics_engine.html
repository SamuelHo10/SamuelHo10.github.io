2D Rigidbody Physics Engine

I developed a 2D rigid-body physics engine from scratch in Java, utilizing the Swing library for graphics rendering. To handle collision detection between convex polygons, I implemented the Separating Axis Theorem (SAT) and used a clipping algorithm to calculate collision points. For collision resolution, I applied an impulse-based approach based on online research and tutorials. The source code for the physics engine, though still largely uncommented, can be accessed here.
