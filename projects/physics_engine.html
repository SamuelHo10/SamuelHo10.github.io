2D Rigidbody Physics Engine

The 2D rigidbody physics engine is coded from scratch in Java with using the swing library for graphics. I implemented the Separating Axis Theorem (SAT) to detect collisions between convex polygons and used a clipping algorithm to determine collision points. For collision resolution, I followed online tutorials that used an impulse-based approach.

Furthermore, for my final project in grade 12 computer science, my group worked on simulating celestial bodies and their orbiting paths in space (using the 2D physics engine). Sadly, I lost the code for that project after switching laptops 😢.
